## Getting Started

### Prerequisites
- Node.js + npm installed
- NASA API key (https://api.nasa.gov/)

### Setup

#### Backend
```
cd backend
npm install
# Put your NASA API key in .env
npm run dev
```

#### Frontend
```
cd frontend
npm install
npm start
```
